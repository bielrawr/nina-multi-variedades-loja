# Nina MULTI Variedades

Site institucional da **Nina MULTI Variedades**, loja localizada em Itabira, MG, voltada para atendimento local com produtos diversos e servicos praticos para o dia a dia.

## Sobre o Projeto

O projeto apresenta a loja, seus principais segmentos de atuacao, informacoes de contato, endereco, horario de funcionamento e links para redes sociais. A proposta e oferecer uma presenca online simples, clara e responsiva, facilitando que clientes conhecam a loja e encontrem rapidamente os canais de atendimento.

## Ramo e Atividades da Loja

A Nina MULTI Variedades atua no comercio varejista com uma selecao ampla de produtos e servicos, incluindo:

- Eletronicos, smartphones, tablets e acessorios;
- Capinhas, peliculas, carregadores e itens relacionados;
- Assistencia tecnica para celulares, incluindo troca de tela e bateria;
- Brinquedos para criancas de diferentes idades;
- Moda e acessorios unissex;
- Artigos de papelaria e materiais escolares;
- Servicos como xerox e solucoes praticas para rotina de estudo e trabalho.

## Funcionalidades

- Pagina inicial com slider de categorias;
- Paginas especificas para Eletronicos, Brinquedos, Moda e Acessorios, e Papelaria;
- Layout responsivo para desktop, tablet e celular;
- Menu mobile com abertura e fechamento acessiveis;
- Secao institucional com descricao da loja;
- Cards com endereco e horario de funcionamento;
- Mapa incorporado do Google Maps;
- Links diretos para WhatsApp e Instagram;
- Rodape responsivo com informacoes de copyright e desenvolvimento.

## Tecnologias Usadas

- **HTML5** para estrutura das paginas;
- **CSS3** para layout, responsividade, animacoes e identidade visual;
- **JavaScript** para controle do menu mobile e slider;
- **Google Fonts** para tipografia;
- **Font Awesome** como apoio para icones;
- **Google Maps Embed** para exibicao do endereco da loja.

## Como Executar Localmente

Por ser um site estatico, basta abrir o arquivo `index.html` no navegador. Tambem e possivel servir a pasta com um servidor local simples:

```bash
python -m http.server 5177
```

Depois, acesse:

```text
http://127.0.0.1:5177/index.html
```
